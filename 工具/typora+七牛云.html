# 准备工作

  1. 电脑先安装typora软件.
  2. 有七牛云账号，并创建一个公共仓库

首先，打开typora软件，进入 文件>>偏好设置>>图像，如下图

  1. 选项改为 无特殊操作
  2. 勾上第一个和第四个框
  3. 上传服务选择 PicGO-Core(command line)
  4. 点击下载或更新按钮

# 配置

下载完之后**PicGO-Core(command line)**点击打开配置文件。参照一下模板更改自己的信息

{
  "picBed": {
    "uploader": "qiniu",
    "qiniu": {
      "accessKey": "",	
      "secretKey": "",	
      "bucket": "", 	
      "url": "", 		
      "area": "", 	
	  "options": "",	  
      "path": "" 		
    }
  },
  "picgoPlugins": {}
}

选项说明：

  • accessKey：AK（密钥管理获取
  • secretKey：SK（密钥管理获取）
  • bucket：仓库名称
  • url：仓库绑定域名，若没有填写仓库自带的
  • area：仓库区域（参考下表）
  • path：仓库路径名称
